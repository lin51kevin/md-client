✨ MarkLite

Lightweight Markdown Editor for Windows · Portable Edition

📂 Quick Start (Portable)

  1. Extract the entire ZIP archive to any folder you like.
  2. Double-click marklite.exe to launch MarkLite.
  3. No installation required! The app runs entirely from its folder.
  4. To uninstall, simply delete this folder.
⚠️ Note: Do not move or rename marklite.exe after extracting. The application expects to run from its extracted directory.

🔧 Optional Setup

If you used the NSIS installer instead of the ZIP package:

  • ☑️ Installed to %LOCALAPPDATA%\MarkLite
  • ☐ Desktop shortcut (optional during install)
  • ☐ Start Menu entry (optional during install)
  • ☑️ File association for .md / .markdown

📌 File Associations

After installing with the NSIS installer, double-clicking any .md file will open it in MarkLite automatically.

If using the portable ZIP version, you can manually associate files: 

assoc .md=MarkLite.md
ftype "MarkLite.md"="C:\path\to\MarkLite\marklite.exe" "%1"
